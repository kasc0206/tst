# 缺失附件自动归类工具

这是一个基于 zotero-plugin-template 改造的 Zotero 7 插件。

## 功能

- 扫描 Zotero 所有可用库（排除 feed 库）中的常规条目
- 识别“有附件但附件文件缺失”的条目
- 将这些条目自动加入名为“缺失附件”的分类（每个库各自维护）
- 若分类不存在会自动创建
- 提供“清理分类”命令：将已恢复附件的条目从“缺失附件”分类移出

## 判定规则

- 仅检查常规文献条目（不检查附件、笔记、注释条目本身）
- 条目没有任何附件时，不视为缺失附件
- 附件类型为“链接 URL”时，不视为本地缺失
- 其他附件若文件不存在，则判定该条目为“缺失附件条目”

## 使用方式

1. 安装依赖：

```bash
npm install
```

1. 构建插件：

```bash
npm run build
```

1. 开发模式运行（热重载）：

```bash
npm start
```

1. 在 Zotero 主界面打开菜单：

Tools -> 扫描缺失附件并归类
Tools -> 清理“缺失附件”分类

1. 扫描完成后会弹出统计信息：

- 扫描条目数
- 识别缺失附件条目数
- 本次新增归类条目数

1. 清理完成后会弹出统计信息：

- 检查条目数
- 移出分类条目数
